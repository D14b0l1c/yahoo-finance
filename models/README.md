# models/

This folder contains trained machine learning models stored in serialized format.

### Files:
- `linear_regression.pkl` - Linear Regression model.
- `random_forest_regressor.pkl` - Random Forest Regressor.
- `xgboost_regressor.pkl` - XGBoost Regressor.
- `arima_forecast.pkl` - ARIMA model for time series.
- `logistic_regression.pkl` - Logistic Regression classifier.
- `lstm_forecast.keras` - LSTM deep learning forecast model.
- `kmeans.pkl` - k-Means clustering model.
- `ebm.pkl` - Explainable Boosting Machine (glass-box regressor).
- `prophet_forecast.pkl` - Prophet time series model.
- `garch_volatility.pkl` - GARCH volatility model.
- `lightgbm_regressor.pkl` - LightGBM Regressor.
- `gru_forecast.keras` - GRU deep learning forecast model.
- `isolation_forest.pkl` - Isolation Forest anomaly detector.
- `portfolio_optimization.pkl` - Mean-Variance optimization results.
